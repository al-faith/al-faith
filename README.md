## who's this

I don't have one lane. If something looks interesting — I poke at it until I understand it, then move to the next thing that catches my eye.

Right now that mostly means **Flutter/Dart**, dipping into **Python for AI tooling**, and keeping **MySQL** around for whenever data needs a home.

No roadmap. No "5-year plan." Just curiosity, doing the driving.

&nbsp;

## the loop

```
see something interesting
        ↓
   get curious
        ↓
 break it apart, rebuild it
        ↓
   keep or discard
        ↓
   repeat (forever)
```
